# Steam on Ubuntu

Valve's Steam client is available to Ubuntu users as a {term}`snap`.

The Steam {term}`snap` provides a consistent environment to run games, which is separate
from the underlying host operating system.

<!-- TODO: the snap also comes bundled with extras, which should be noted -->

It can be run on any system supporting {term}`snapd`, including Ubuntu.
